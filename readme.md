# 11ty with Architect

Following https://www.filamentgroup.com/lab/build-a-blog/

## Build:

```bash
npm run build
```

## Run locally:

```bash
npm start
```

# Setup CI/CD:

Sign into Begin.com and click **Create new app** &rarr; **Import an app**  &rarr; `brianleroux/arc-example-11ty`

## Deploy to staging:

Commit to Github! 

## Deploy to production:

Tag a relase!
